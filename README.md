# WildWatch - Endangered Species Data Website

A comprehensive, visually rich website showcasing endangered animals with conservation data, stunning photography, and interactive features.

## 🌍 Features

- **Rich Animal Gallery** - High-quality photographs of endangered species
- **Conservation Dashboard** - Real-time statistics on species populations
- **Interactive Species Cards** - Detailed information for each animal
- **Threat Analysis** - Visual breakdown of major threats to wildlife
- **Success Stories** - Conservation wins and population recoveries
- **Habitat Map** - Global biodiversity hotspot visualization
- **Latest News** - Conservation updates and breakthroughs
- **Get Involved** - Ways to help protect endangered species

## 🚀 Deployment Instructions

### Option 1: Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files (index.html, style.css, script.js, README.md)
3. Go to Settings → Pages
4. Select "main" branch and "/ (root)" folder
5. Click Save
6. Your site will be live at: `https://[username].github.io/[repository-name]/`

### Option 2: Local Development

1. Download all files to a folder
2. Open index.html in your browser
3. No server required - works offline!

## 🎨 Design Highlights

- **Earth-tone color palette** - Greens, browns, and warm accents
- **Card-based layout** - Easy to browse species information

The site automatically adapts to:
- Desktop (1200px+)

## 🔧 Customization

To add more species:
1. Open `script.js`
2. Add new object to `speciesData` array
3. Include: name, scientific, status, population, region, image URL
4. Save and refresh
